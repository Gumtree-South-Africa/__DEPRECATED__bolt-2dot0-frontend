# bolt-2.0-frontend Server

bolt-2.0-frontend projects with NodeJS Platform


## Dev Mode

```
git clone git@github.corp.ebay.com:ecg-global/bolt-2dot0-frontend.git
cd bolt-2dot0-frontend
npm run dev
```

## Quick deploy

```
git clone git@github.corp.ebay.com:ecg-global/bolt-2dot0-frontend.git
cd bolt-2dot0-frontend
npm run build
DEBUG=bolt-2dot0-frontend:* npm start
```

Then access:
http://localhost:8000

# Bolt 2.0 Handlebars

Handlebar template inheritance

## Usage

'''handlebar
//block
{{block "SFO"}} {{/block}}

// static parital
{{> "SFO" }}

// dynamic partial
{{> (dynamic "SFO") }}

// base partial 
{{> (base "SFO")}}
'''

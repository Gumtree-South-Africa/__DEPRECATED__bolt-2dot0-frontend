# Handlebar Helpers

Handlebars Helpers

### Prerequisities

Make sure package.json got the laterest version of express-handlebar

### Useage
```handlebars

{{i18n "my.name is %s. i'm %s old. I live in, %s" "anton" "20" "santa cruz"}}
 
{{i18n "my.name is %s. i'm %s old." "anton" "20" }}
 
{{i18n "my.name is %s." "anton"  }}
 
 {{i18n "my.name" }}
```
